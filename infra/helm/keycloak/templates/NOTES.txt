 Access
------------------------------------------------------------------------

{{- if .Values.ingress.enabled }}
Keycloak is exposed via Ingress:

{{- range .Values.ingress.hosts }}
  URL: https://{{ .host }}
{{- end }}

Ensure DNS records point to the Ingress controller's external IP.
TLS is {{ if .Values.ingress.tls }}enabled{{ else }}disabled -- configure ingress.tls for production{{ end }}.
{{- else }}
Keycloak is only accessible inside the cluster.
To reach it from your workstation, run:

  kubectl port-forward svc/{{ include "xiam-keycloak.fullname" . }} 8080:{{ .Values.service.port }} \
    -n {{ .Release.Namespace }}

Then open http://localhost:8080 in your browser.
{{- end }}

------------------------------------------------------------------------
 Admin Console
------------------------------------------------------------------------

The Keycloak admin console is available at:

  /admin/master/console/

Admin credentials are stored in Secret "{{ include "xiam-keycloak.adminSecretName" . }}".

To retrieve the admin password:

  kubectl get secret {{ include "xiam-keycloak.adminSecretName" . }} \
    -n {{ .Release.Namespace }} \
    -o jsonpath='{.data.KEYCLOAK_ADMIN_PASSWORD}' | base64 -d
